# Project Brief (Persistent)
- **Vision produit:** CryptoSense aide a comprendre, comparer et analyser des crypto-actifs long terme de maniere structuree, pedagogique et neutre, sans conseil financier.
- **Conventions:** JS standard, pas de TypeScript, pas de DB en V1
- **Qualite:** QA manuel obligatoire avant deploy
- **Commandes cles:** npm run dev / build / start
- **Cadence de mise a jour:** a chaque changement majeur d'architecture ou scope
